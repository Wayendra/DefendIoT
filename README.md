# DefendIoT
Final Year Research Project
An Advanced Intrusion Detection System with real-time monitoring, device management, and security analytics Designed and Built Mainly and Specifilically for IOT Devices, with a C2 Specific Module.

##Overview
DefendIoT IDS is a web-based intrusion detection system built with React, TypeScript, and multiple real-time backends (WebSocket, SSE, and Socket.IO). It provides separate experiences for super admins and client users, along with device monitoring, alert triage, and reporting.

Key Features
Dual-role access control for super admins and client users
Real-time alerts via WebSocket (alerts stream)
Live table updates via SSE for users, clients, devices, and endpoints
IoT device onboarding and status management
Remote terminal view for device operators
CSV exports and report generation
PostgreSQL persistence (ThreatSense and User DBs)
